
# Linux Administration Lab

This lab demonstrates core Linux system administration skills using Ubuntu Server 22.04.

## Projects
- Server hardening (SSH key auth, UFW, fail2ban)
- User onboarding automation (create_user.sh)
- Backup automation (backup-etc.sh + cron)
- System reporting (sys-report.sh)
- Simple containerized app (Podman + Nginx)

## How to run
See `docs/setup-instructions.md` for step-by-step lab build and `scripts/README.md` for script usage.
